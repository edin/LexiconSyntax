# Lexicon Syntax Docs

Lexicon Syntax is a compact grammar language for describing tokens, parser
rules, typed AST nodes, metadata attributes, and generation settings for
Lexicon-based parsers.

Start here:

- [Language](language.md) explains `.lxs` grammar files.
- [CLI And Config](cli-and-config.md) explains `lsyn` commands and config
  resolution.
- [Generation](generation.md) explains generated token enums, AST nodes, and
  recursive parsers.
- [Validation](validation.md) explains current semantic checks.
- [Roadmap Notes](roadmap-notes.md) collects design directions that are not
  fully implemented yet.

The repository includes a C-like grammar example under `examples/`.
